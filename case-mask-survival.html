Mask Survival

A stealth survival game — collect four Oni masks from a Monster House while evading faction-aware enemy AI.

// 01

Purpose

Mask Survival was built as a 12-week solo project to prove end-to-end game development capability — from AI architecture and gameplay systems to audio production and shipping on a public platform.

The core design challenge was building a stealth game where enemy AI felt genuinely threatening without being frustrating. Each enemy needed to belong to a faction with distinct behavioral rules, respond to both sight and sound, and coordinate with other enemies on alert — all within Unreal Engine 5's Blueprint system.

// 02

Implementation

The project was built entirely in UE5 Blueprints with a focus on modular, reusable systems.

  • 01
    Faction-Aware Enemy AI Behavior Tree architecture with per-faction patrol, chase, and alert states. Each enemy type belongs to a faction that shares alert status — when one sees the player, nearby allies escalate.
  • 02
    AI Perception System Dual-sense perception using UE5's AIPerception component — sight cone for visual detection and hearing radius for sound events. Per-actor handling via OnTargetPerceptionUpdated with bool guards to prevent duplicate broadcasts.
  • 03
    Throwable Decoy System Hearing decoy using ReportNoiseEvent with bool guards. Players throw decoys to redirect AI patrol paths, creating strategic windows for movement.
  • 04
    Struct-Based Mask Inventory Multi-mask inventory using a struct array. Resolved an enum index offset bug and stale array index issues discovered during debugging. Each mask carries unique properties driving gameplay state changes.
  • 05
    NavMesh & Movement Non-latent AIMoveTo fix via BTService + Blackboard Vector key. Debugged three distinct NavMesh failures including dynamic obstacle recalculation issues.
  • 06
    Audio Pipeline Original soundtrack produced in Ardour/Audacity using LANDR samples across five tracks. Implemented via CreateSound2D stored in the Game Instance for persistent audio across level loads.
  • 07
    Packaging & Shipping Resolved shader compiler hang on packaging by disabling Lumen/Nanite and clearing DDC cache. Fixed packaged build Open Level failure. Shipped to itch.io with full build verification.

Takeaway

Mask Survival proved I can own the full development loop — designing systems, debugging complex AI interactions, producing original audio, and shipping a packaged build to a public platform solo and on deadline.

The most significant technical growth came from debugging multi-system interactions: AI Perception conflicts, NavMesh failures, and packaging errors that only surface in a shipped build. Each one required reading UE5 internals rather than surface-level fixes.

The project directly informs Yoruba Hearts — the faction-aware AI architecture, Behavior Tree patterns, and modular inventory design from Mask Survival are the foundation I'm scaling for the larger action RPG system.
